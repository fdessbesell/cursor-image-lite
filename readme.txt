Cursor Image Lite
=================

Plugin leve para personalizar o cursor do site com imagens PNG.
- Cursor padrão (obrigatório)
- Cursor hover (opcional) para links/botões
- Oculta o cursor padrão e usa imagem personalizada em desktop
- Não mostra cursor personalizado em dispositivos móveis (touch)
- Imagens devem ser PNG e até 200KB (recomendado)
